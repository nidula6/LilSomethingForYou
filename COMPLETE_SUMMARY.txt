🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Music Won't Play:
  1. Check: Is music/song.mp3 there? (Yes - already in folder)
  2. Click: Play button (auto-play requires user interaction)
  3. Try: Different browser (Chrome/Firefox preferred)
  4. Check: Press F12 and look for console errors

Letter Won't Open:
  1. Click: On the envelope directly
  2. Check: JavaScript is enabled
  3. Try: Hard refresh (Ctrl+Shift+R)

Animations Not Working:
  1. Try: Chrome or Firefox (best support)
  2. Clear: Browser cache (Ctrl+Shift+Delete)
  3. Check: JavaScript is enabled

Changes Not Showing:
  1. Edit: script.js and save
  2. Refresh: Page (F5 or Ctrl+R)
  3. Hard Refresh: Ctrl+Shift+R to clear cache
